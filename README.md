# Gelişmeler

Bu depo, yazılım öğrenme sürecimde geliştirdiğim küçük projeleri ve alıştırmaları içermektedir.

## Amaç

Programlama dillerini öğrenirken yaptığım çalışmaları düzenli olarak saklamak ve gelişimimi takip etmek.

## İçerik

Şu anda:

- Python alıştırmaları
- Algoritma çalışmaları
- Temel fonksiyon örnekleri
- ...

bulunmaktadır.

İlerleyen zamanlarda aşağıdaki teknolojilerle ilgili çalışmalar da eklenecektir:

- HTML
- CSS
- JavaScript
- C#
- Arduino
- Diğer yazılım projeleri
- ...

## Not

Bu depodaki projeler eğitim ve öğrenme amaçlıdır. Kodlar zamanla geliştirilecek ve yeni projeler eklenecektir.
